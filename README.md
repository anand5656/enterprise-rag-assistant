# 🚀 Production-Grade Enterprise GenAI RAG Assistant

An enterprise-style AI Knowledge Assistant built using **FastAPI**, **FAISS**, **Groq LLM**, and **Sentence Transformers** with Retrieval-Augmented Generation (RAG).

This project allows users to upload documents (PDF/DOCX/TXT), perform semantic retrieval, and generate context-aware AI responses with source citations.

---

# 📌 Features

✅ Retrieval-Augmented Generation (RAG)

✅ Dynamic PDF / DOCX / TXT Upload

✅ Semantic Search using FAISS

✅ Groq LLM Integration

✅ Conversation Memory

✅ Enterprise-Style Chat UI

✅ Source Citations

✅ Chunk-Based Document Retrieval

✅ FastAPI Backend

✅ Responsive Frontend

✅ Production-Ready Project Structure
